MySQL Project – UMKM Jawa Barat



Tujuan dari tugas ini adalah melakukan eksplorasi dan analisis data UMKM di Jawa Barat menggunakan query SQL, mulai dari basic query untuk pengolahan data sederhana hingga advanced query untuk analisis mendalam dengan agregasi dan klasifikasi.



Basic Query

Fungsi yang digunakan:

* SELECT untuk menampilkan data.
* WHERE untuk filter kota, tahun, kategori usaha.
* ORDER BY untuk mengurutkan data.
* LIMIT untuk membatasi jumlah baris.
* DISTINCT untuk menampilkan kategori unik.
* IN, OR, NOT IN untuk filter multi-kategori.
* LIKE untuk pencarian pola pada kode kabupaten/kota.

Contoh Analisis

* Menampilkan data UMKM di Kota Bandung.
* Menyaring UMKM kategori Makanan dan Fashion.
* Melihat tren UMKM kategori Batik di Kabupaten Tasikmalaya (2018–2020).
* Menampilkan UMKM kuliner di wilayah Bandung tahun 2021.

Hasilnya adalah memberikan gambaran awal tentang distribusi UMKM berdasarkan lokasi, kategori usaha, dan tahun.



Advanced Query

Fungsi yang digunakan:

* COUNT, SUM, AVG untuk agregasi data.
* GROUP BY untuk analisis per kategori usaha/tahun.
* HAVING untuk filter hasil agregat.
* MIN, MAX untuk mencari nilai ekstrem.
* CASE WHEN untuk klasifikasi data (tinggi/rendah).

Contoh Analisis

* Menghitung jumlah UMKM di Kabupaten Bekasi tahun 2017.
* Melihat tren UMKM di Kabupaten Karawang (2017–2021).
* Mengidentifikasi kabupaten/kota dengan UMKM < 100.000 pada tahun 2020.
* Menentukan kategori usaha dengan jumlah UMKM terbanyak dan tersedikit di Kabupaten Sukabumi tahun 2020.
* Membuat klasifikasi jumlah UMKM (Tinggi/Rendah) dengan CASE WHEN.

Hasilnya adalah memberikan insight lebih mendalam tentang tren UMKM, rata-rata per kategori, distribusi antar kabupaten/kota, serta klasifikasi jumlah UMKM.
