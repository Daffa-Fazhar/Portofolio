Project – Hackathon



Tujuannya adalah membangun simulasi bisnis kos-kosan dengan Python berdasarkan modal awal dan aturan tambahan dari file Excel. Output berupa data bulan-ke-bulan yang menampilkan pembangunan kamar, pendapatan sewa, pengeluaran, serta penyesuaian aturan.



Proses

1. Input Data
modal.xlsx -> saldo awal, biaya bangun, harga sewa, threshold.
informasi-tambahan.xlsx -> aturan perubahan (harga sewa, biaya bangun, threshold, stop building).
2. Simulasi
Script simulation.py membaca kedua file, menerapkan aturan, lalu menghasilkan simulasi.xlsx.
3. Output

&#x20;  Sheet per individu (Susi, Budi, Siti, Bambang) berisi timeline keuangan dan jumlah kamar, 

&#x20;  plus sheet Harta untuk saldo akhir dan total kamar.



Hasil

* Menunjukkan bagaimana strategi berbeda memengaruhi pertumbuhan kamar dan saldo.
* Memberikan insight tentang dampak perubahan harga sewa, biaya bangun, dan threshold terhadap investasi kos-kosan.
