Project 3 – Pivot Table Sederhana (Airline Analysis)



Pada project ini saya mengolah data penerbangan menggunakan kombinasi rumus IF, VLOOKUP, HLOOKUP, SUM, RIGHT, TIME sebelum membuat pivot table.

* Tujuan: Menyajikan analisis sederhana terkait pendapatan per rute, kelas penerbangan, status bagasi, dan durasi rata-rata.
* Metode: Menggunakan fungsi lookup untuk pencarian data, Menggunakan fungsi logika IF untuk klasifikasi, Menggunakan pivot table untuk merangkum pendapatan dan distribusi bagasi.
* Hasil: Dashboard sederhana yang menampilkan total biaya, kelas terlaris, rata-rata durasi penerbangan, serta distribusi bagasi per rute. Visualisasi ini membantu memahami performa rute dan preferensi penumpang
