Project 1 – Analisis Data dengan Fungsi Logika \& Lookup



Pada project ini saya menggunakan berbagai fungsi Excel untuk melakukan analisis data secara terstruktur. Fungsi yang digunakan meliputi: IF, IFS, INDEX MATCH, XLOOKUP, VLOOKUP, HLOOKUP, OR, AND, serta Conditional Formatting.

* Tujuan: Membuat sistem analisis yang dapat mengklasifikasikan data, mencari nilai secara dinamis, dan menyoroti kondisi tertentu.
* Hasil: Data lebih mudah dipahami, pencarian nilai lebih akurat, dan visualisasi tabel lebih informatif berkat conditional formatting.
