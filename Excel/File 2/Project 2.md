Project 2 – Analisis Data dengan Fungsi Teks \& Perhitungan



Project ini berfokus pada pengolahan data berbasis teks dan perhitungan waktu.

Fungsi yang digunakan meliputi: DATEDIF, IF, IFS, MID, LEFT, RIGHT, INDEX MATCH, VLOOKUP, HLOOKUP, SUMIF, CONCATENATE.

* Tujuan: Menghitung selisih waktu, mengolah string teks, serta melakukan pencarian dan agregasi data.
* Hasil: Data dapat diolah menjadi lebih rapi, informasi waktu lebih jelas, dan laporan lebih ringkas dengan fungsi penggabungan teks.
