Project 4 – Pivot Table Kompleks (Sales Dashboard)



Project ini lebih kompleks karena melibatkan beberapa sheet: Top Penjualan, Rekap Provinsi, Rekap Sales, Channel, Day, dan Order.

* Tujuan: Membuat sistem analisis penjualan yang komprehensif, rapi, dan mudah dipahami melalui pivot table multi-sheet.
* Metode: Menggunakan pivot table untuk setiap sheet agar data lebih terstruktur, Menggabungkan hasil pivot menjadi dashboard interaktif dengan filter (tahun, bulan, item type, channel), Menyajikan metrik utama: order priority, unit sold, revenue, cost, profit, serta distribusi penjualan offline/online.
* Hasil: Dashboard kompleks yang menampilkan peta penjualan per provinsi, persentase order priority, penjualan harian per channel, serta provinsi dengan penjualan tertinggi. Visualisasi ini mempermudah analisis performa penjualan lintas wilayah dan kategori produk.
